# Array Sorted Checker (C Program)

## 📌 Description
This program checks whether an array is sorted in ascending order.

## 🚀 How It Works
- Compare each element with the next element.
- If any element is greater than the next one, the array is not sorted.
- Otherwise, the array is sorted.

## 📥 Input Array

```text
10 20 30 40 50
```

## 📤 Output

```text
Sorted
```

- Arrays
- Loops
- Conditional Statements
- Flag Variable
